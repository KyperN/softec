# SERVER

## install

### requirements

node version: 16.15.1
npm version: 8.1.2

create .env file like in .env-example

```shell script
# install dependencies
npm i

# run server
npm run dev
```

# CLIENT

## install

### requirements

react version: 18.2.0

create .env file like in .env-example

```shell script
# install dependencies
npm i

# run server
npm start
```
